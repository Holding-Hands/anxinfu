# 🎨 主题换肤使用说明

## 功能位置

在页面**右上角导航栏**，用户头像的左侧，有一个**紫色渐变圆形按钮**（带太阳图标）。

## 使用方法

1. **点击紫色圆形按钮** - 会弹出主题选择菜单
2. **选择你喜欢的主题** - 点击任意主题即可切换
3. **立即生效** - 页面会立即应用新主题，无需刷新

## 8 种主题

| 主题 | 颜色标识 | 效果 |
|------|---------|------|
| 🔵 默认蓝 | 蓝色圆点 | 经典蓝白配色 |
| 🌙 暗黑夜 | 灰色圆点 | 深色护眼模式 |
| 💜 优雅紫 | 紫色圆点 | 紫色侧边栏 |
| 🌿 清新绿 | 绿色圆点 | 绿色侧边栏 |
| 🟠 活力橙 | 橙色圆点 | 橙色侧边栏 |
| 💗 浪漫粉 | 粉色圆点 | 粉色侧边栏 |
| 🔷 商务蓝 | 深蓝圆点 | 深蓝侧边栏 |
| ❤️ 热情红 | 红色圆点 | 红色侧边栏 |

## 主题效果

切换主题后，以下内容会改变：
- ✅ 侧边栏背景色和文字颜色
- ✅ 子菜单背景色（完全匹配主题色）
- ✅ 页面背景色
- ✅ 顶部导航栏
- ✅ 所有按钮和交互元素
- ✅ 表格、表单等组件
- ✅ 卡片和容器

## 如何查看效果

建议操作：
1. 点击按钮打开主题菜单
2. 选择 **💜 优雅紫** - 看到紫色侧边栏
3. 展开 **盟友管理** - 看到子菜单也是紫色
4. 尝试其他主题，观察颜色变化

## 常见问题

**Q: 看不到换肤按钮？**
A: 按钮在页面右上角，紫色渐变圆形，带太阳图标

**Q: 点击没反应？**
A: 请确保开发服务器正在运行，刷新页面重试

**Q: 主题没有变化？**
A: 请检查浏览器控制台是否有错误信息

## 技术说明

- 使用 Element Plus Popover 组件
- 基于 CSS 变量实现动态主题
- 自动保存到 LocalStorage
- 支持 8 种预设主题
