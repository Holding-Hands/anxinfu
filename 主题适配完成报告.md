## 🎨 使用的变量系统

### Element Plus 原生变量

所有组件现在都使用 Element Plus 的原生 CSS 变量：

```scss
// 颜色
--el-color-primary
--el-color-success
--el-color-warning
--el-color-danger
--el-color-info

// 背景
--el-bg-color
--el-bg-color-page
--el-fill-color-blank
--el-fill-color
--el-fill-color-light

// 文字
--el-text-color-primary
--el-text-color-regular
--el-text-color-secondary
--el-text-color-placeholder

// 边框
--el-border-color
--el-border-color-light
--el-border-color-lighter
--el-border-color-extra-light
```

## 🚀 测试验证

### 测试步骤

1. **刷新浏览器**（Ctrl + F5 强制刷新）
2. 点击右上角换肤按钮
3. 选择"暗黑夜"主题
4. 检查以下内容：

#### 表格检查

- [ ] 表格背景是深色的（不是白色）
- [ ] 表头背景稍深一些
- [ ] 表格边框是深色的
- [ ] 鼠标悬停时行背景变化明显

#### 分页器检查

- [ ] 页码按钮背景是深色的
- [ ] 点击"15条/页"下拉框，列表是深色背景
- [ ] 当前页高亮显示（蓝色背景）

#### 下拉框检查

- [ ] "实名状态"、"用户类别"等下拉框背景是深色
- [ ] 打开下拉列表，列表背景是深色
- [ ] 悬停选项时有高亮效果

#### 按钮检查

- [ ] "查询"按钮（蓝色）正常显示
- [ ] "重置"按钮（默认）背景是深色
- [ ] "导出"按钮（绿色）正常显示
- [ ] 悬停按钮时有交互效果

#### 卡片检查

- [ ] 筛选卡片背景是深色
- [ ] 表格卡片背景是深色
- [ ] 卡片边框颜色协调

- **清新绿**: 绿色主题，所有组件跟随

- **浪漫粉**: 粉色主题，所有组件跟随

## 📝 技术说明

### 优势

1. **使用 Element Plus 原生变量**: 确保与组件内部样式完全一致
2. **!important 优先级**: 确保主题样式优先于默认样式
3. **全面覆盖**: 所有状态（hover、focus、active、disabled）都已适配
4. **自动适配**: 新增的 Element Plus 组件也会自动跟随主题

### 注意事项

- 所有修改都在 `src/styles/themes.scss` 中
- 使用了 SCSS 嵌套语法提高可读性
- 每个组件都有明确的注释
